## Getting Started

Run with:

```bash
npx create-next-ts-starter
```

It contains default setup for projects.
Main technologies used:

- React
- Next.js
- Typescript
- Tailwind.css
- Docker
- NextPWA
- Sass

All the config for eslint, prettier, husky and typescript is provided.

Avalible commands:

```bash
npm start
npm run check-types
npm run check-format
npm run check-lint
npm run format
npm run test-all
```
